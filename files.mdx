Upload documents for your agent to reference.

## Supported Formats

| Format | Extensions |
|--------|------------|
| PDF | `.pdf` |
| Word | `.docx` |
| Text | `.txt`, `.md` |
| Excel | `.xlsx`, `.xls`, `.csv` |

## Upload

Drag and drop files or click to browse. Multiple files supported.

## Visual Processing

Enable **Visual Processing** checkbox before uploading PDFs with:
- Screenshots or diagrams
- Complex tables
- Scanned documents

<Note>
Visual processing is slower but extracts content from images.
</Note>

## Upload Status

| Status | Meaning |
|--------|---------|
| Queued | Waiting to process |
| Processing | Being indexed |
| Complete | Ready to use |
| Failed | Check error message |

## Managing Files

View all uploaded files in the document list. Click delete to remove a file (cannot be undone).

## Tips

- Keep files under 50MB for fastest processing
- Use descriptive filenames
- Remove unnecessary pages before upload
- Test in Sandbox after adding content
